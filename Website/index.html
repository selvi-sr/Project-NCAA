NCAA Basketball - Project 3

ABOUT

PROJECT MADNESS

We chose to utilize the Men's NCAA Basketball Kaggle dataset for this project.

The original data set was quite large as it included a wide variery of stats for the 68-team national championship from 2003 thru 2018. We chose to limit our analysis range to 2013 thru 2018 in order to provide better predictability in our model. We additionally found that while the dataset gave information on teams in the championship, we were unable to drill down to an individual player level for a shot analysis. We did have to use an additional NBA dataset to find similarities in player shots.
